# ⚔️ Python Feudal RPG Engine

A modular, text-based RPG engine built in Python. This project focuses on state management, branching narratives, and resource-efficient game logic.

## 🚀 Features
- **Dynamic State Machine:** Tracks player choices and environmental changes across different sessions.
- **Modular Room System:** Easily expandable map architecture using JSON-based data loading.
- **Inventory & Stats Logic:** A robust system for managing player attributes and item interactions.

## 🛠️ Technical Implementation
- **Language:** Python 3.x
- **Architecture:** Procedural logic with a focus on clean, readable code and separation of concerns (Logic vs. Data).
- **Optimization:** Designed to run with minimal overhead, suitable for low-spec environments.

## 🎮 How to Play
1. Clone the repository:
   `git clone https://github.com/YOUR_USERNAME/your-repo-name.git`
2. Run the game:
   `python main.py`
